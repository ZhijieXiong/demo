# 1、如何划分结构

- 自上而下、从左到右、以行为单位
- 划分的标准：内容、颜色块、间距、边框

# 2、样式重置

- `reset.css`的作用就是样式重置
  - html的原生标签带了许多CSS的样式
  - 样式重置的作用就是将无用的原生样式去除

# 3、样式的模块化

- 通过模块化加载`reset.css`
- CSS的`@规则`
  - `@charset`    设置样式表的编码
  - `@import`    导入其它样式文件
  - `@media`    媒体查询（移动端）
  - `@font-face`    自定义字体（如使用自定义图标时需要用到）

# 4、三栏布局

- 假如要求三个独立板块排列在一行，如何实现？
- 让左边的板块左浮动，右边的板块右浮动，然后设置中间的板块居中显示即可
- 居中显示：`margin: 0 auto;`

# 5、关于结构嵌套

- 尽量少嵌套结构，嵌套深度越大，搜索引擎越不好爬取其中内容
